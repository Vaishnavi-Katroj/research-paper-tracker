# Research Paper Tracker (Python Tkinter)

A GUI-based Research Paper Tracker built using Python and Tkinter that helps users manage research papers efficiently.  
The application allows users to add, view, and delete research paper records with persistent CSV file storage.

## Features
- Add research papers with title, authors, domain, and status
- View all saved papers in a tabular format
- Delete selected research paper entries
- Data persistence using CSV file storage
- User-friendly graphical interface
- Input validation and warning messages

## Technologies Used
- Python
- Tkinter (GUI)
- CSV file handling
- OS module for file management

## How to Run
1. Make sure Python is installed on your system
2. Download or clone this repository
3. Open terminal in the project folder
4. Run the application using:
```bash
python research_paper_tracker.py
